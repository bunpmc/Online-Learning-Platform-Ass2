# Main User Workflows

This document outlines the primary workflows of the LearnHub platform, reflecting the refined logic implemented in Phase 4.

## 1. Skill Assessment & Personalized Recommendations

1. Navigate to `/Assessment/Start`.
2. Select your experience level, interest areas (e.g., Web Development), and learning goals.
3. Observe that selecting an answer **does not** scroll the page to the top.
4. Click "Complete Assessment".
5. View the Assessment Result page. Recommendations should be sorted by **Match Score** based on your selected interests.

## 2. Learning Path Discovery & Enrollment

1. Navigate to `/LearningPath/Browse`.
2. Notice that Learning Paths you haven't purchased yet **do not** show progress bars or journey stats.
3. Click on a Learning Path to see its details.
4. Click "Enroll Now". You should be taken **directly** to the Checkout page.
5. Complete the purchase.
6. Verify that you are now enrolled in the Path **and** all individual courses within that path.

## 3. Course Learning & Progress

1. Go to "My Learning" ( `/Course/MyCourses`).
2. Verify only individual courses you have purchased (or got via a Path) are shown.
3. Click "Continue Learning" on a course.
4. Watch a video (simulated) and click "Mark as Complete".
5. Observe the progress bar at the top of the learning interface updates in real-time.
6. Complete all lessons and verify the "View Certificate" button appears.
